# @repo/ui

Shared UI component library for Remindrop.

Built with:
- shadcn/ui
- Radix UI
- TailwindCSS
- TypeScript

## Usage

```tsx
import { Button } from '@repo/ui/components/button';
import '@repo/ui/styles/globals.css';

export function MyComponent() {
  return <Button>Click me</Button>;
}
```
